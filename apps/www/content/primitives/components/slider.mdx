## Props

The `Slider` component accepts the following props:

- `variant`: The type of slider (`"single"` | `"range"`, default: "single")
- `min`: Minimum value (default: 0)
- `max`: Maximum value (default: 100)
- `step`: Step increment (default: 1)
- `defaultValue`: Initial value(s) - number for single, [number, number] for range
- `value`: Controlled value(s) - number for single, [number, number] for range
- `label`: Optional label text to display above thumb(s)
- `onChange`: Callback when value changes
- `aria-label`: Accessibility label
- All other HTMLDivElement attributes from React

## Accessibility

The Slider component implements the following accessibility features:

- Uses `role="slider"` for proper semantic meaning
- Provides appropriate `aria-label` based on variant and labels
- Supports keyboard navigation
- Shows focus indicators for keyboard users
- Reports current values via `aria-valuenow`
- Sets proper min/max ranges with `aria-valuemin` and `aria-valuemax`

Example with custom aria labels:
