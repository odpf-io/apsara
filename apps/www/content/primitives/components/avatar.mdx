## Usage
The Avatar component utilizes a masked image when the source (src) is provided. 
It functions as a visual embellishment, assuming that the accompanying textual description is available from nearby elements. 
However, in cases where there is no textual content accessible, 
it is recommended to include an aria-label attribute to ensure screen-reader accessibility.
